Projects and templates

A project in Code on the Go contains everything that defines your workspace for an app, including source code and resources to help test and build your app. Code on the Go contains several project templates that automatically create the right structure for your project files.

Create a project

To learn about the different kinds of projects you can create in

To create a project, follow these steps on the Get started page:

1.Tap Create project. The New project page is displayed.

2.Select a template. (Learn more in About project templates, below.) A configuration page for the project is displayed.

3.In the App name field, enter a name for your app. The Package name field will automatically be updated to include the name of your app.

4.The project will be saved in the default directory. To change the directory, tap Save location and select a different directory.

5.Tap Create project. Code on the Go will create new project files and directories. Please be patient while Code on the Go sets up the project. When the process is finished, Project Initialized will be displayed at the bottom of the screen.

To see the file structure for your app and begin writing code, swipe from the left side of the screen to the right. The file tree for your project will be displayed.
Open a saved project

To open a saved project, follow these steps on the top Code on the Go menu page:

1.Tap Open a saved project. A screen to locate the saved project is displayed.

2.If the displayed list of folders includes the AndroidIDEProjects folder, tap AndroidIDEProjects to open the folder.

3.Navigate into the project folder. You must be able to see the top level of files to select the folder. Show example

Delete a project

WARNING: Deleting a project permanently removes all of the project's files and directories. A deleted project can't be restored.

To delete a project, follow these steps on the Get started page:

1.Tap Delete a project. SOMETHING WILL BE DISPLAYED.

2.Open the folder of the project you want to delete. This will be rewritten when we have a normal file manager.

3.Confirm etc

NOTE: Jira re not deleting the AndroidIDEProjects folder.

About project templates

A project template provides the blueprint for a certain type of app. Code on the Go uses the template you choose to create the structure of the project and the files to build your app. Project templates also provider starter code to help you create your app faster.

Code on the Go provides templates for the following project types:

Project type

About the template

No activity

This template creates a completely blank project with no automatically generated files. If you use this template, you must manually create all the files that your app needs, including the MainActivity file.

Empty

This template creates a simple MainActivity file. The default app contains no navigation or interaction elements.

Show/hide the default empty project app

Basic

This template creates a MainActivity file and a floating action button.

Show/hide the default basic project app

Navigation drawer

This template creates a MainActivity file with a floating action button, the name of the home page,  and a left-hand navigation drawer.  The navigation drawer is an overlay panel that covers some or all of an activity's screen and usually contains all the options that are available in the app. To display the left-hand navigation drawer, swipe left to right or click the three-bar menu at the upper left.

Jetpack Compose

Jetpack Compose is a modern Kotlin-based Android UI toolkit for building interfaces using Google's Material Design (a set of visual design guidelines).  Jetpack Compose typically requires less code than traditional XML layouts, simplifies testing and debugging the UI, and can improve app performance.

To use Jetpack Compose, you must use Kotlin. Java is not supported.
