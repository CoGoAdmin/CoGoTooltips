Overview: Building an Android app with Code on the Go

Here's a high-level overview of building an Android app using Code on the Go. Click any item to see more information.

1. Start a new project

•Choose a project template, such as Empty Activity, which provides a basic layout for your app with one screen.

•Define your app name, package name, and target device API level.
2. Understand the project structure

•Java/Kotlin code: Located in the app/src/main/java directory. This is where you'll write your program logic, such as activities, services, etc.

•XML layouts: If you're not creating a Jetpack Compose project, XML layouts for the app user interface (UI) are located in the app/src/main/res/layout directory.

•Manifest file: AndroidManifest.xml defines essential app information, like activities, permissions, and app metadata.
3. Create the user interface (UI)

•Use the Layout Editor (drag-and-drop) or write XML to define your app's UI.

•Common elements include TextView, Button, and RecyclerView.
4. Write Java or Kotlin code (logic)

•Logic for the app is written in activities such as MainActivity.java (or other activities).

•Lifecycle methods such as onCreate(), onStart(), and onDestroy() control how the app behaves as it moves between different states. Learn more about the Android lifecycle.

5. Run and test the app

•Use the Android Emulator (or connect a real device) to run and test your app.

•CAN YOU REALLY? You can simulate different screen sizes, versions, and orientations.
6. Debug the app

•Code on the Go has a built-in debugger to help you find and fix issues.

•You can use breakpoints and view logs using...
7. Build the app

•On the main project page, tap the green arrow. The icon will change to red circles while the app is being built.

•If the app builds successfully, a prompt to install the app is displayed.

•If the app is not not built, the BUILD FAILED message is displayed at the bottom of the screen. To troubleshoot the app, tap BUILD FAILED. The Build Output log is displayed.

Start a new project

1.On the home screen, tap Create Project. A page of project templates is displayed.

2.Select the template for your project. Learn more about project templates. The page to enter the app's name, programming language and SDK is displayed. A configuration page for the project is displayed.

3.In the App name field, enter a name for your app. The Package name field will automatically be updated to include the name of your app.

4.The project will be saved in the default directory. To change the directory, tap Save location and select a different directory.

5.Tap Create project. Code on the Go will create new project files and directories. Please be patient while Code on the Go sets up the project. When the process is finished, Project Initialized will be displayed at the bottom of the screen.

To see the file structure for your app and begin writing code, swipe from the left side of the screen to the right. The file tree for your project will be displayed.
