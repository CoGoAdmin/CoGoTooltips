Documentation and reference materials

Get help anywhere

Three kinds of help are available throughout Code on the Go.

•Tooltips. To display a tooltip, long-press anything on the screen. A short tooltip will be displayed.

•Expanded tooltips. Some tooltips provide additional information without leaving the Code on the Go screen. If a + button is visible on the tooltip, tap + to display additional help.

•Comprehensive documentation of features. Links to related documentation are located at the bottom of the tooltip window.

oTo display the documentation, click a link.

oTo return to Code on the Go screen from the documentation, use the phone or tablet's Back button.
