Search projects and files

You can search, summary stuff.

Find in current file

Only in text editor.

1.Tap the Search icon.

2.Tap Find in File. At the bottom of the screen, the Text to search field is displayed. Show Text to search field (highlighted)

3.Enter the text to search for. As soon as you enter text, additional options will be displayed below the field.

NOTE: If you want to use regex to search for regular expressions, enter the regular expression to search for.

4.Choose one of the following:

•To search for the exact text you entered, including upper and/or lower case characters, tap NEXT. The next occurrence of the text in the file is highlighted.

•To ignore upper and lower case in the search term, tap the overflow menu and select Ignore case. Then, tap NEXT.

•To use regex, tap the overflow menu and select Use regex and then tap NEXT.

Find and replace in current file

1.Tap the Find icon.

2.Enter text.

3.Tap REPLACE. The Replace box is displayed.

4.In the Replacement field, enter the text you want to use instead of the search term.

5.Choose one of the following:

•To replace the currently selected text, tap Replace. The text will be changed and the search window will close.

•To replace all instances of the text, tap Replace All.

To undo the replacement, tap the Undo button.  

Find in all project files

JIM, if you search in all project files, you can't choose to ignore case or use regex. Does that matter?

1.In the main project window, tap the Find icon and select Find in project. (If no files are open, this is only option.) The Find in project dialog box is displayed.

2.In the Search by modules, select one or more modules. You must select at least one module.

3.Enter the text you want to search for.

4.If you want to organize your results by file extensions, enter each extension followed by the | character. (Example:  HUH? 

5.Tap Find. The Search results tab in the output panel will open and display the results of your search.

6.Tap a result to open that file.
